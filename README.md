# Family Resonance — The First Spark

Soul constellation visualization. Enter any Quantum Signature to see verified bonds, frequency match scores, and lineage echoes around it.

**Live:** https://family.thefirstspark.shop

## How it works

1. Each soul has a **Quantum Signature** — a deterministic 12-char hash generated by `impossible_code.py` in `thefirstspark/soul-map-server`
2. Bonds between souls are stored in `bonds.json` with verified status and resonance scores
3. The page renders a constellation centered on the input signature with all bonds orbiting

## Data model

- `souls` — registry of every signature with name, life path, frequency, rarity
- `bonds` — `{ soul_a, soul_b, relationship, context, verified, resonance, type, dna_confirmed }`

## Bond types (color-coded)

- **Bloodline** (#FF6A3D ember): parent_child, siblings, partners, extended_family
- **Soul Resonance** (#6B4DF2 violet): frequency_twin, soul_resonance
- **Ancestor / Lineage** (#26E4D8 cyan): ancestor
